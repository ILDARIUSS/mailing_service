# 📧 Django Mailing Service

Сервис управления и автоматической отправки email-рассылок.

## 🚀 Возможности проекта

- Создание рассылок через Django Admin
- Отправка писем по расписанию
- Автоматическая отправка через планировщик задач (Windows Task Scheduler)
- Разделение ролей пользователей (администратор / пользователь)
- Хранение конфигурации через переменные окружения

---

## ⚙️ Технологии

- Python 3.13
- Django
- SQLite
- SMTP (Yandex)
